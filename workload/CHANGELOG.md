## [Unreleased]

### Исправлено
- «Форма нагрузки» теперь правильно считает часы, когда один учитель ведёт обе группы предмета с делением: в жёлтом столбце, в разбивке по классам и в общем итоге учителя. Раньше такая нагрузка учитывалась только за одну группу — чаще всего это касалось кафедры информатики
- Классы в «Форме нагрузки» теперь правильно отображаются в формате «5-а» — убран лишний дефис для имён классов, уже записанных через дефис в учебном плане (было «5--а»)

- Блок «Вакансии» теперь группирует по таблице кафедры, а не по кафедре целиком: «География: Вакансия Г. — 25 ч/нед» вместо «ХимБиоГео: …»

## [1.19.1] — 2026-04-17

### Изменено
- Блок «Вакансии» на вкладке «Экспорт» теперь показывает только имя учителя и количество часов в неделю — без детализации по предметам и классам
- Имя файла Word/PDF отражает следующий учебный год: нагрузка, составленная в 2026, скачивается как «нагрузка_учителей_2026-2027»

## [1.19.0] — 2026-04-16

### Добавлено
- На вкладке «Экспорт» появился жёлтый блок «Вакансии» — показывает, на какие предметы и классы назначен учитель-заместитель, ожидающий замены

### Изменено
- Убрано предупреждение «близко к лимиту СанПиН» — остался только сигнал об явном превышении
- На вкладке «Назначения» обязательные предметы всегда отображаются перед выборными

## [1.18.0] — 2026-04-16

### Добавлено
- На вкладке «Экспорт» предупреждения теперь раскрываются: нажмите на баннер, чтобы увидеть полный список
- На вкладке «Экспорт» появилось зелёное сообщение «Всё в порядке» — когда нагрузка полностью соответствует учебному плану

### Изменено
- В учебном плане обязательные предметы всегда отображаются перед выборными — порядок соответствует структуре исходного Excel независимо от ручных правок

## [1.17.3] — 2026-04-13

### Исправлено
- Исправлена ложная ошибка «превышение СанПиН» для классов, у которых нагрузка по учебному плану ровно на уровне лимита: «Разговоры о важном» теперь не учитываются в академическом лимите

### Добавлено
- При превышении СанПиН теперь показывает расшифровку: какие предметы входят в нагрузку и сколько часов каждый

## [1.17.2] — 2026-04-13

### Добавлено
- В ZIP-архив релиза добавлен файл «Прочитай!.txt» с инструкцией по запуску

## [1.17.1] — 2026-04-13

### Улучшено
- Плановый технический аудит обоих приложений: устранены выявленные расхождения в логике, упрощена внутренняя структура кода
- Внутренняя оптимизация: страницы «Назначения» и «Учебный план» теперь перерисовываются только при изменении нужных данных хранилища
- Добавлены тесты для экспорта «Формы нагрузки» в HTML и Word

## [1.17.0] — 2026-04-10

### Изменено
- Кнопка «Печатать нагрузку» убрана со страницы «Экспорт» — она устарела; для получения официального документа используйте «Скачать Word» или «Скачать PDF»
- В «Форме нагрузки» (Word/PDF) предметы теперь группируются по кафедрам и идут в том же порядке, что и в настройках кафедр; каждая кафедра выделена зелёной строкой-заголовком в таблице

## [1.16.1] — 2026-04-09

### Исправлено
- Классы в таблицах «Назначений» и «Учителей» теперь всегда отсортированы по параллели: 5-е идут первыми, 10-е и 11-е — в конце

## [1.16.0] — 2026-04-08

### Добавлено
- На странице «Учебный план» появилась кнопка «Скачать шаблон .xlsx» — скачивает готовый файл с примером структуры и инструкцией
- На странице «Учебный план» появилась кнопка «Скачать УП (.xlsx)» — скачивает текущий учебный план в формате Excel для оформления официального документа
- Классы теперь сортируются по номеру параллели (5, 6, ..., 10, 11) — десятые и одиннадцатые классы больше не перемешиваются с пятыми
- Список занятий в экспорте теперь сортируется по классам
- На вкладке «Классные руководители» теперь можно вводить фамилию для быстрого поиска учителя
- «Проверить нагрузку» теперь предупреждает, если на предмет назначено больше учителей, чем предусмотрено планом

### Улучшено
- На вкладке «Назначения» красная подсветка в заголовках классов теперь показывает превышение по предметам данной таблицы, а не по всему учебному плану
- Импорт файла кафедры больше не меняет структуру кафедры завуча (названия таблиц, привязки предметов) — из снимка берутся только новые учителя и назначения
- При импорте файла кафедры появляется предупреждение, если в списке учителей обнаружены возможные дубликаты (одинаковые имена с разными ID)

### Исправлено
- Ложные ошибки превышения СанПиН по классам — итоги в заголовках столбцов на вкладке «Назначения» теперь правильно считают каждый предмет один раз, даже при делении на группы

## [1.15.1] — 2026-03-31

## [1.15.0] — 2026-03-27

### Добавлено
- Назначения: кнопка «Отменить все» в заголовке столбца класса — снимает все назначения данного класса в таблице одним кликом; действие отменяется через Ctrl+Z
- Кнопка «Очистить данные…» в меню сохранения — сбрасывает все данные до начального состояния после подтверждения

### Исправлено
- Импорт файла кафедры больше не блокируется при расхождении версий учебного плана: назначения принимаются в любом случае; если часть из них не совпадает с текущим планом, они удаляются автоматически и отображается подробная информация о том, что именно нужно назначить заново

## [1.14.0] — 2026-03-22

### Добавлено
- Форма нагрузки: автоматическая генерация официального документа «Нагрузка учителей» в форматах Word (.docx) и PDF; поле «Вариант» с датой и меткой; компактная запись классов «5-а,б,в(6)»; составные предметы (Рус+Лит, Алг+Геом+Вер) с разбивкой в жёлтой ячейке; раздел элективов; итоговые строки

### Улучшено
- Назначения: таблица теперь скрывает столбцы классов, у которых нет предметов для данной группы кафедры — лишние пустые столбцы больше не занимают место

## [1.13.0] — 2026-03-21

### Улучшено
- Деление предмета на группы: чекбокс по-прежнему применяет изменение ко всем параллелям, но рядом добавлена кнопка «только эту» — для случаев когда нужно указать разные настройки в разных параллелях (например, физкультура: не делится в 1–9 классах, делится в 10–11)
- Десктопная версия (macOS/Windows): при закрытии окна с несохранёнными данными появляется диалог «Сохранить и выйти / Выйти без сохранения / Отмена»; кнопка «Сохранить» открывает системный диалог выбора файла вместо автосохранения в Downloads
- Релизный пайплайн: сборки для macOS (.dmg) и Windows (.exe) теперь создаются автоматически при каждом релизе
