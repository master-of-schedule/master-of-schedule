## Что это такое

В репозитории два самостоятельных приложения, которые работают вместе:

**РШР** — редактор расписания. Сюда загружается список занятий из Excel, а дальше уроки расставляются по сетке вручную или перемещаются перетаскиванием. Готовое расписание можно скопировать в Google Таблицы или скачать файл для импорта в журнальную систему.

**Редактор нагрузки** — отдельный инструмент, с которого всё начинается. Завуч загружает учебный план, распределяет предметы между учителями по кафедрам и выгружает файл занятий — тот самый Excel, который потом идёт в РШР.

## РШР

<img src=".github/assets/screenshot-main.png" alt="Редактор расписания" width="100%" />

Расписание класса — слева список классов, в центре сетка уроков (предмет, учитель, кабинет), справа нераспределённые занятия.

### Что умеет

- Загружает данные из Excel или JSON-файла
- Три типа версий: **шаблон** (журнал), **техническое** и **на неделю**
- Жёлтая подсветка ячеек, где недельное расписание отличается от шаблона
- Проверяет конфликты: занятость учителя, запреты по времени, вместимость кабинетов
- Параллельные группы с автоматическим определением пар
- Замены: отмечать отсутствующих учителей, предлагать замещающих
- Скачать PNG-картинку с изменениями — для отправки в мессенджер
- Архив прошлых учебных лет
- Экспорт в Google Таблицы (TSV) и в систему электронного журнала

## Редактор нагрузки

<img src=".github/assets/screenshot-workload.png" alt="Редактор нагрузки" width="100%" />

Таблица назначений — учителя кафедры по строкам, классы по столбцам. Зелёный заголовок — класс укомплектован полностью.

### Что умеет

- Парсит учебный план из Excel
- Распределяет предметы между учителями в разрезе кафедр
- Считает нагрузку по СанПиН и лимиту 34 ч/нед, подсвечивает превышение
- Классные руководители + «Разговоры о важном» автоматически
- Печатает сводку нагрузки по кафедре
- Экспортирует `занятия.xlsx` для импорта в РШР

## Быстрый старт

Готовые сборки — на странице [Releases](../../releases). Все данные хранятся локально, подключения к интернету не требуется.

### Скачать готовую сборку

| Файл | Для чего |
|------|----------|
| `РШР-x64.dmg` | Десктоп-приложение macOS |
| `РШР-x64-setup.exe` | Десктоп-приложение Windows |
| `РШР-web-vX.Y.Z.zip` | РШР в браузере — распакуйте и откройте `index.html` |
| `РН-web-vX.Y.Z.zip` | Редактор нагрузки в браузере |

### Запустить локально

```bash
git clone https://github.com/Vanderhoof/timetable.git
cd timetable

# РШР
npm install
npm run dev          # http://localhost:5173

# Редактор нагрузки (в отдельном терминале)
cd workload
npm install
npm run dev          # http://localhost:5174
```

Для десктоп-версии РШР через Tauri (требуется [Rust](https://rustup.rs)):

```bash
npm run tauri:dev
```

## Технологии

- **React + TypeScript + Vite** — оба приложения
- **Zustand** — управление состоянием
- **Dexie (IndexedDB)** — хранилище данных в РШР
- **Tauri 2** — десктоп-обёртка для РШР
- **xlsx** — чтение/запись Excel-файлов
- **vite-plugin-singlefile** — сборка в один HTML-файл
