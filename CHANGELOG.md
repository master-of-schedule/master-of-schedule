# История изменений — РШР

Все значимые изменения документируются здесь.
Формат: [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)

---

## [Не выпущено]

### Добавлено
- В панели «Занятия» (недельное расписание): при отметке «Проведено» теперь выбирается количество проведённых занятий — 1 или 2; проведённые занятия отображаются в конце списка с зачёркиванием и значком ✓
- Поддержка классов партнёрской школы: в таблице классов и в форме редактирования появился чекбокс «Партнёр»; классы партнёра отображаются в конце списка навигации со сдвинутым приоритетом; конфликты по учителю в классах партнёра отображаются серым (не оранжевым); при экспорте расписания классы партнёра исключаются из JSON занятости
- В редакторе появилась кнопка «Отменить JSON партнёра» — убирает загруженный файл занятости партнёрской школы

### Улучшено
- Убрана ссылка на функцию «Двойной клик — автокабинет» из подсказки в редакторе (функция не реализована)
- В режиме «Шаблон» теперь отображаются панели «Учитель» и «Кабинет»; раздел «Другие (проф.)» в панели замен скрыт
- Пункт «Больничный» убран из контекстного меню панели «Занятия»; проведённые занятия исключены из поиска замен

## [3.30.0] — 2026-04-03

### Добавлено
- Раздел «Другие (проф.)» в панели замен теперь свёрнут по умолчанию — раскрывается по клику, показывает количество доступных учителей
- Клик по записи в протоколе изменений теперь восстанавливает расписание до этого состояния напрямую
- На странице «Главная» появилась кнопка «Шаблон Excel» — скачивает пустой `.xlsx` с нужными листами и заголовками для заполнения данных

## [3.29.1] — 2026-03-31

### Исправлено
- В панели «Заменить» раздел «Напарники в этом уроке» теперь отображается выше раздела «Другие (проф.)»
- Изображение с профсоюзными заменами теперь имеет заголовок «Профсоюз», а не «Замены»
- Номер версии теперь отображается в заголовке окна и диалоге «О программе»

---

## [3.29.0] — 2026-03-30

### Добавлено
- При создании занятия через «+» появился выпадающий список «Тип»: `—` / `Замена (бюджет)` / `Замена (проф.)` — показывается только в недельной версии; выбранный тип автоматически попадает в журнал замен
- В панели «Заменить» появился раздел «Напарники в этом уроке» — при клике на напарника открывается окно создания занятия с уже заполненными учителем и предметом

### Исправлено
- Ctrl+Z и Ctrl+Y теперь работают при включённой русской раскладке клавиатуры; при исчерпанной истории показывают подсказку «Нечего отменять» / «Нечего повторить»
- Фамилии учителей нормализуются при сохранении и импорте Excel — устраняет возможное искажение символов при определённых настройках Chrome
- При автосохранении «Скачать для мессенджера» профсоюзные замены теперь сохраняются в ту же папку «Замены для завуча», что и бюджетные
- Кнопки «Сохранить» и «Открыть» убраны из заголовка — их аналоги есть на странице «Главная». Кнопка с версией теперь хорошо видна

## [3.28.0] — 2026-03-29

### Добавлено
- Номер версии в шапке приложения — при нажатии открывается окно «О программе» с именами авторов
- Панель «Кабинет» в редакторе — выбрать кабинет и день, увидеть все занятия в этом кабинете, отметить нужные; отображается рядом с панелью «Учитель»
- Панель «Отсутствует» переименована в «Учитель»
- При выборе замены появился раздел «Другие (проф.)» — свободные учителя, не ведущие этот предмет; замена помечается как профсоюзная
- В экспорте появилась отдельная кнопка «Замены (проф.)» для скачивания профсоюзных замен
- «Скачать для мессенджера» теперь сохраняет в четыре папки одновременно: картинки, замены, полный JSON, JSON занятости; кнопка ⚙ рядом открывает диалог настройки папок

### Исправлено
- Заверённые нарушения запретов перестали сбрасываться при изменениях в расписании — теперь подтверждение сохраняется до закрытия или смены версии

## [3.27.0] — 2026-03-21

### Улучшено
- В десктопной версии кнопка «Сохранить» теперь открывает нативный диалог выбора папки и имени файла — файл больше не пропадает в Downloads без спроса

## [3.26.0] — 2026-03-20

### Исправлено
- Заверение нарушений запретов теперь сохраняется между сеансами: закрой программу, открой снова — заверение на месте, повторно подтверждать не нужно
- В десктопной версии при закрытии с несохранёнными изменениями теперь появляется диалог с тремя вариантами: «Сохранить», «Закрыть без сохранения», «Отмена»

### Добавлено
- В десктопной версии в шапке появилась кнопка выхода — небольшая, иконкой

### Улучшения
- При импорте Excel без листа «Учителя» программа сама составит список учителей из списка занятий — с предметами, без запретов

## [3.25.0] — 2026-03-19

### Исправлено
- Первый урок в новом расписании теперь можно отменить через Протокол
- Исключённые из проверки окон классы теперь сохраняются при экспорте и импорте файла
- Групповые окна теперь определяются корректно — данные о парах групп берутся из таблицы групп
- При закрытии exe-версии теперь появляется предупреждение о несохранённых изменениях

### Добавлено
- При создании шаблона или расписания теперь сразу предлагается ввести название
- Режим копирования теперь остаётся активным после вставки — занятие можно расставить сразу в несколько ячеек
- Excel-файл теперь принимается даже если лист «Учителя» пустой
