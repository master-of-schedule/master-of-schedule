ЧТО НАХОДИТСЯ В ЭТОЙ ПАПКЕ
───────────────────────────
Файл index.html — это и есть программа. Она работает прямо
в браузере, без установки.


КАК ОТКРЫТЬ ПРОГРАММУ
──────────────────────
Не открывайте index.html двойным кликом — он может открыться
не в том браузере или показать пустую страницу.

Правильный способ:
  1. Щёлкните по файлу index.html ПРАВОЙ кнопкой мыши.
  2. Выберите «Открыть с помощью».
  3. Выберите Microsoft Edge или Google Chrome.


КАКОЙ БРАУЗЕР ПОДХОДИТ
───────────────────────
  ✓ Microsoft Edge — рекомендуется
  ✓ Google Chrome — рекомендуется

  ✗ Яндекс.Браузер — некоторые функции могут работать
                      неправильно
  ✗ Internet Explorer — программа в нём не работает


СОВЕТ: ЧТОБЫ ПРОГРАММА ОТКРЫВАЛАСЬ САМА
────────────────────────────────────────
В Microsoft Edge можно настроить восстановление вкладок —
тогда программа будет открываться автоматически при следующем
запуске браузера.

Как включить:
  1. Откройте Edge, нажмите «···» в правом верхнем углу.
  2. Выберите «Параметры» → «Запуск».
  3. Включите «Открывать вкладки, которые были открыты
     в прошлый раз».

После этого просто не закрывайте вкладку с программой —
при следующем открытии браузера она будет на месте.
